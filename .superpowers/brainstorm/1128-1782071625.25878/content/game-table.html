iPhone 横屏适配要点

🏷 安全区 — 上44px / 下44px / 右40px / 左60px（Dynamic Island 避让）

🪑 座位 — 围绕桌面分布的玩家卡片需要 responsive 定位。6座时：顶2、底1（当前玩家）、左右中3空位。默认用百分比定位 + `transform: translate(-50%,-50%)` 居中

🃏 公共牌 — 5张牌始终横排居中，`left:50%` + `translateX(-50%)`，不随屏幕宽度偏移

🎮 动作栏 — 底部 fixed，横排按钮 + raise 金额 −/+，left/right 对齐 safe area

📱 竖向空间紧张 — 1320px 竖向高度要容纳：顶栏 ~80px + 座位层 ~300px + 公共牌 ~120px + 底池 ~40px + 当前玩家 ~120px + 动作栏 ~60px + 安全区上下 88px ≈ 还剩 ~500px 空间。足够。
